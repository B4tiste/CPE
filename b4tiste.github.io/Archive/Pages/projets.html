Robot Croupier
Projet en collaboration avec Vahe Djorkaeff

Projet effectué dans le cadre de l'obtention du DUT Génie Electrique et Informatique Industrielle.
Le but de ce projet était de créer un robot qui jouait le rôle du croupier lors d'une partie de Blackjack.
Le robot était capable de reconnaître les cartes distribuées, grâce à une caméra et à des algorithmes de reconnaissance d'image.

Technologies utilisées :
  • Python
    • OpenCV
  • Linux
  • RaspberryPi / PiCam
  • Modélisation 3D
